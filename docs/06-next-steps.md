---

### Phase 2 · Build session 1 (2–3 hours)
The core site comes together.

**What we build:**
1. Initialize the Astro project
2. Set up design tokens (colors, fonts, spacing) from the creative brief
3. Build the homepage — all 13 sections — in their initial form with placeholder content where needed
4. Build the basic page shells for Services, About, Process, Contact
5. Wire up the navigation
6. Commit to GitHub
7. Deploy to Cloudflare Pages (staging URL)

**Output:** A live, viewable site at a `pages.dev` URL. Not polished. Not final. But real, deployable, and editable.

---

### Phase 3 · Build session 2 (2–3 hours)
Polish, content, and finalization.

**What we build:**
- Finalize homepage copy (headline, all section text, testimonial attributions)
- Build out the 4 interior pages with real copy
- Drop in real client logos and Kristin's headshot
- Final design review and refinement (spacing, colors, animations)
- SEO pass (page titles, meta descriptions, structured data)
- Accessibility pass (alt text, keyboard navigation, color contrast)
- Generate Privacy Policy and Terms of Service pages
- Set up Cloudflare Web Analytics
- Connect the `soloex.io` domain

**Output:** A complete, launchable site on your real domain.

---

### Phase 4 · Launch and iterate
You go live. We keep improving.

**Launch day:**
- Announce on LinkedIn
- Update Upwork profile to link to the new site
- Update email signature
- (If ready) Publish AppExchange listing with the site link

**First 30 days:**
- Monitor Cloudflare analytics for traffic patterns
- Collect feedback from early visitors and clients
- Add new client logos as permission comes in
- Collect 2–3 more testimonials
- Draft 1–2 case studies for a future Work page

**First 3 months:**
- Consider adding a Resources/Tools page (lead magnets)
- Consider starting a newsletter (if list-building becomes a priority)
- Refine copy based on which messaging resonates
- Add any features you hit the need for (e.g., a blog, a pricing calculator)

- **The goal for day 1 is better than Framer.** You don't need to match a $100k agency design. You need a real, working, AI-editable site that represents Soloex correctly.
